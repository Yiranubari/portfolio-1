# Yiranubari Promise — Portfolio

Personal portfolio site built with React (no TypeScript).

## Getting Started

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Build for Production

```bash
npm run build
```

Outputs to the `build/` folder. Drop it on Netlify, Vercel, or any static host.

## Deploy to Netlify (easiest)

1. Run `npm run build`
2. Go to [netlify.com/drop](https://app.netlify.com/drop)
3. Drag and drop the `build/` folder
4. Done — live in seconds
